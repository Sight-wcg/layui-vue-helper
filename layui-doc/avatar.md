### Avatar Attributes

| 属性   | 描述               | 类型      | 默认值 | 可选值         |
| ------ | ------------------ | --------- | ------ | -------------- |
| src    | 图源               | `string`  | -      | --             |
| size   | 尺寸               | `string`  | -      | `xs` `sm` `lg` |
| radius | 圆形               | `boolean` | -      | `true` `false` |
| alt    | 描述图像的替换文本 | -         | -      | --             |
