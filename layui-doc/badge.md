### Badge Attributes

| 属性  | 描述 | 类型     | 默认值 | 可选值                                        |
| ----- | ---- | -------- | ------ | --------------------------------------------- |
| type  | 类型 | `string` | -      | `dot` `rim`                                   |
| theme | 主题 | `string` | -      | `orange` `green` `cyan` `blue` `black` `gray` |

### Badge Slots

| 插槽    | 描述 | 备注          |
| ------- | ---- | ------------- |
| default | 默认 | 非 `dot` 可用 |
