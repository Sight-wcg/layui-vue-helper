### configProvider Attributes

| 属性          | 描述             | 类型     | 默认值  | 可选值          |
| ------------- | ---------------- | -------- | ------- | --------------- |
| locale        | 地区             | `string` | `zh_CN` | `zh_CN` `en_US` |
| locales       | 地区语言包       | `object` | -       | -               |
| theme         | 主题模式         | `string` | `light` | `light` `dark`  |
| themeVariable | 主题变量         | `object` | -       | -               |
| darkPartial   | 调整夜间模式表现 | `object` | -       | -               |

### configProvider Slots

| 名称    | 说明     | 参数 |
| ------- | -------- | ---- |
| default | 默认插槽 | -    |
