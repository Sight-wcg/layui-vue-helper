### Exception Attributes

| 属性     | 描述     | 类型     | 默认值 | 可选值            |
| -------- | -------- | -------- | ------ | ----------------- |
| title    | 标题     | `string` | -      | --                |
| status   | 状态     | `string` | -      | `403` `404` `500` |
| describe | 描述信息 | `string` | -      | --                |
