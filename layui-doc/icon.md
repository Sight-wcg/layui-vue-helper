### Icon Attributes

| 属性   | 描述 | 类型     | 默认值       | 可选值 |
| ------ | ---- | -------- | ------------ | ------ |
| type   | 图标 | `string` | -            | -      |
| prefix | 前缀 | `string` | `layui-icon` | -      |
| color  | 颜色 | `string` | --           | -      |
| size   | 尺寸 | `string` | --           | -      |
