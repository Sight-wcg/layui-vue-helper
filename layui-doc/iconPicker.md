### Icon Picker Attributes

| 属性       | 描述     | 类型      | 默认值 | 可选值         |
| ---------- | -------- | --------- | ------ | -------------- |
| v-model    | 默认值   | string    | -      | --             |
| page       | 开启分页 | `boolean` | -      | `true` `false` |
| showSearch | 启用搜索 | `boolean` | -      | `true` `false` |
