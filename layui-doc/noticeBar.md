### Notice Bar Attributes

| 属性       | 描述             | 类型      | 默认值  | 可选值             |
| ---------- | ---------------- | --------- | ------- | ------------------ |
| mode       | 模式             | `string`  | `link`  | 'link' 'closeable' |
| text       | 滚动内容         | `string`  | -       | -                  |
| leftIcon   | 左侧图标         | `string`  | --      | -                  |
| rightIcon  | 右侧图标         | `string`  | --      | -                  |
| scrollable | 是否开启垂直滚动 | `boolean` | `false` | true,false         |
| textlist   | 垂直滚动内容     | string[]  | -       | -                  |
