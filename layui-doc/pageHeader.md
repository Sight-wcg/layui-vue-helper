### Page Header Attributes

| 属性     | 描述                          | 类型     | 默认值 | 可选值 |
| -------- | ----------------------------- | -------- | ------ | ------ |
| content  | 标题,当前默认插槽不为空时失效 | `string` | --     | --     |
| backText | 返回文本                      | `string` | "返回" | --     |

### Page Header Events

| 属性 | 描述         | 可选值 |
| ---- | ------------ | ------ |
| back | 点击返回回调 | -      |
