### Scroll Attributes

| 属性       | 描述     | 类型     | 默认值 | 可选值 |
| ---------- | -------- | -------- | ------ | ------ |
| height     | 高度     | `string` | --     | --     |
| thumbWidth | 滑块宽度 | `string` | --     | --     |
| thumbColor | 滑块颜色 | `string` | --     | --     |
| trackColor | 滑槽颜色 | `string` | --     | --     |
