### Slider Attributes

| 属性       | 描述         | 类型      | 默认值  | 可选值         |
| ---------- | ------------ | --------- | ------- | -------------- |
| v-model    | 选中值       | `Number`  | -       | -              |
| vertical   | 是否垂直     | `Boolean` | `false` | `true` `false` |
| range      | 是否区间     | `Boolean` | `false` | `true` `false` |
| rangeValue | 区间值       | `Array`   | -       | -              |
| step       | 步长         | `Number`  | -       | -              |
| min        | 最小值       | `Number`  | -       | -              |
| max        | 最大值       | `Number`  | -       | -              |
| showDots   | 是否显示断点 | `Boolean` | `false` | `true` `false` |
