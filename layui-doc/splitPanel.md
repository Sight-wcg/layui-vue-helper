### Split Panel Attributes

| 属性     | 描述                        | 类型      | 默认值  | 可选值         |
| -------- | --------------------------- | --------- | ------- | -------------- |
| vertical | 是否垂直布局                | `Boolean` | `false` | `true` `false` |
| minSize  | 块拉动最小范围(按像素 `px`) | `number`  | -       | `50`           |

### Split Panel Item Attributes

| 属性  | 描述                                                                          | 类型   | 可选值 | 默认值       |
| ----- | ----------------------------------------------------------------------------- | ------ | ------ | ------------ |
| space | 默认每个站多大比例（`%`）, 设置一个，<br/> 其他的都需要设置，合计为（`100` ） | number | -      | 按照个数平分 |
