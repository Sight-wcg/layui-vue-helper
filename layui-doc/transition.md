### Transition Attributes

| 属性   | 描述           | 类型      | 默认值     | 可选值             |
| ------ | -------------- | --------- | ---------- | ------------------ |
| enable | 启用           | `boolean` | `true`     | `true` `false`     |
| type   | 类型           | `string`  | `collapse` | `collapse`, `fade` |
| time   | 自定义过渡时常 | -         | -          | -                  |
